# Technologies used:
- Node.js
- Nest.js
- dotenv
- typeORM
- Postgres

# How to run:
- Clone repo with "git clone"
- Install packages with "npm i"
- Create new PostgreSql database and specify credentials for it
- Create .env file, example in ".env.example"
- Run "npm run migrations:run"
- Run "npm run start:dev" to start in dev mode
